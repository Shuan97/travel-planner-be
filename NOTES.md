## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## File generation

```bash
# module modules/<name>
$ nest g module modules/name

# component modules/<name>
$ nest g co modules/name

# service modules/<name>
$ nest g s modules/name

#
$

```
